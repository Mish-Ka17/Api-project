## О проекте Api-project

Выгрузка данных по указанным эндпоинтам (Продажи - /api/sales, Заказы - /api/orders, Склады - /api/stocks, Доходы - /api/incomes) осуществляется посредством консольной команды (app:import-command), вызываемой из папки api-project в формате: <p>php artisan app:import-command --from=Дата начальная --to=Дата конечная --page=Номер страницы --limit=Количество записей на странице,</p>
<p>где:</p>
    <p>Дата начальная и Дата конечная - строки вида Y-m-d (Например, 2026-06-27),</p>
    <p>Номер страницы - целое число (по умолчанию 1),</p>
    <p>Количество записей на странице - целое число (по умолчанию 500 - максимальное значение).</p>

<p>Дата начальная и Дата конечная (--from и --to) - обязательные параметры.</p>
<p>Для выгрузки данных по Складам по умолчанию берется текущая дата вне зависимости от указанных Дата начальная и Дата конечная.</p>
<p>Номер страницы и Количество записей на странице (--page и --limit) - необязательные параметры (по умолчанию 1 и 500, соответственно).</p>
<p>Пример вызова консольной команды: </p>
<p>php artisan app:import-command --from=2026-06-24 --to=2026-06-26 --page=2 --limit=200</p>
<p>Примечание: выгрузка данных по Доходам (/api/incomes) не осуществляется ввиду того, что API не выдает данные по этому эндпоинту</p>
